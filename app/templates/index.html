LinkedIn Share Assist

Personal Project

This is a terminal application meant to be a fire and forget auto posting program. First it extracts job opportunities from the Dollar Bank Career website, then it filters and formats the data to be posted to LinkedIn, under the users personal account. Several configuration files help control all these behaviors and the automated process, but the program also includes a manual mode. This project allowed me to practice and familiarize my self with the following technologies:

  • Python
  • LinkedIn API
  • Sublime, Vim
  • Bash, Unix, Linux, Ubuntu
  • CRON
  • smtp servers
